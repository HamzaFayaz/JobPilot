# CV templates (supported layout)

These files match the layout JobPilot’s suggested-CV editor is optimized for.

| File | Use |
|------|-----|
| `JobPilot-CV-Template.docx` | Download, fill, upload to Profile |
| `JobPilot-CV-Template.pdf` | Preview only (structure reference) |

Upload to the live server with **Actions → Upload CV templates** (manual).  
Public URLs after upload: `/downloads/JobPilot-CV-Template.docx` and `.pdf`.
